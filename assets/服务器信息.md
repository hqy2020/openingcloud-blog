## HTTPS 证书

| 项目 | 值 |
|------|-----|
| 域名 | openingclouds.ccwu.cc |
| 证书颁发机构 | Let's Encrypt |
| 申请方式 | Certbot webroot 验证 |
| SSL 协议 | TLSv1.2 / TLSv1.3 |
| 自动续期 | certbot 容器每 12 小时检查一次 |

### 证书路径（容器内）

- 证书链: `/etc/letsencrypt/live/openingclouds.ccwu.cc/fullchain.pem`
- 私钥: `/etc/letsencrypt/live/openingclouds.ccwu.cc/privkey.pem`

### 宿主机映射

- `./certbot/conf` → `/etc/letsencrypt`（证书存储）
- `./certbot/www` → `/var/www/certbot`（ACME 验证目录）

### 相关文件

- 初始化脚本: `scripts/init_https_certbot.sh`
- HTTPS Nginx 配置: `nginx/nginx.https.conf`
- Docker Compose 叠加: `docker-compose.https.yml`

### 首次申请命令

```sh
./scripts/init_https_certbot.sh openingclouds.ccwu.cc your-email@example.com
```

### 端口

- `:80` — HTTP，自动 301 跳转至 HTTPS（保留 ACME challenge 路径）
