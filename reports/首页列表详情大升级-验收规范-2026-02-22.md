# 首页/列表/详情大升级验收规范(轻量版)

- 版本: v0.1
- 日期: 2026-02-22
- 覆盖范围: `/`、`/tech|/learning|/life`、`/posts/:slug`
- 说明:
  - 本文档是“本次大升级”的轻量验收规范，不替代 `reports/验收清单.md` 与 `reports/用户体验验收清单.md`。
  - 以“功能与关键交互可用”为主，视觉细节不做阻断级全覆盖。

## 1. 首页(`/`) 验收

### P0: 封面 Hero(最重要)

- [ ] 文案结构存在且可读: `My name is:` + 名字(间隔字母) + 角色/定位 + CTA。
- [ ] 太阳系公转 Logo 可见且持续动画存在(公转+自转)。
- [ ] reduced-motion 下: 3D/大动效显著降级，但封面仍完整且不空白。
- [ ] WebGL 不可用时: Hero 有静态 fallback(轨道/图片/渐变)，控制台不报错。
- [ ] “浙大/阿里云”徽章可见(含 icon + stats)，不遮挡主标题。
- [ ] CTA 可用: 主 CTA 能进入文章列表；次 CTA 能跳到项目区或 Time 区。

### P0: 高光成就双向跑马灯

- [ ] 两行卡片跑马灯方向相反(上左下右或上右下左)，内容集合一致。
- [ ] hover 任一卡片时该行滚动暂停；离开恢复滚动。
- [ ] 卡片内容可读(标题/时间/描述)，不出现“滚动导致无法阅读”的问题。

### P0: Web Coding 项目区(含指向技术栈)

- [ ] hover 项目卡有明显反馈(抬升/聚光/轻 tilt)。
- [ ] hover 项目卡时出现“指向技术栈 chips”的指示线/指针，且对应 chips 高亮。
- [ ] 点击项目卡能跳转到项目链接(新窗口或同窗口策略一致)。

### P0: Time 堆叠面积图(qzq.at 复刻形态)

- [ ] 图表存在: 堆叠面积图渲染成功。
- [ ] x 轴为 Age(年龄)，y 轴为 Percent(0%~100%)。
- [ ] hover 某区域可见描边高亮效果(加粗/变亮均可)。

### P1: 地图(旅行足迹 + 当前雷达)

- [ ] 旅行点位有涟漪/脉冲效果，tooltip 或信息提示可用(如有)。
- [ ] 当前所在位置有更强雷达闪现(视觉强度明显高于普通旅行点)。
- [ ] 地图加载失败时有降级展示(列表/卡片)，页面不白屏。

### P1: 双雷达图

- [ ] 雷达图 A/B 都可渲染，维度数量为 6 或 7 且与配置一致。
- [ ] hover 有 tooltip(或明确降级为表格仍可读)。

### P1: Dock

- [ ] 桌面端 Dock 可见且可交互(至少 5 个入口)。
- [ ] Dock 不遮挡正文关键内容(尤其在滚动到页面底部时)。
- [ ] 移动端策略符合 PRD(隐藏或替代)，不影响核心 CTA。

## 2. 列表页(`/tech|/learning|/life`) 验收

- [ ] Tabs 切换分类正常，动效不卡顿。
- [ ] 排序 `最新/热度(views)` 生效，列表顺序发生变化。
- [ ] Tag 过滤写入 URL，刷新页面仍保持筛选结果。
- [ ] 自动加载更多可用: 不重复、不抖动，错误时有提示。
- [ ] 工具条可展开/收起(Expandable Toolbar 形态)，移动端不挤压布局。

## 3. 文章详情页(`/posts/:slug`) 验收(移动端 Tabs)

- [ ] 小屏幕出现 Tabs: `文章/目录/信息`(命名可微调但语义一致)。
- [ ] 切换 Tabs 不白屏、不丢关键内容。
- [ ] 目录 Tab 可跳转到对应标题，定位偏差可接受(不超过一个标题高度)。
- [ ] 信息 Tab 包含至少: 标签、阅读时长、阅读量或更新时间。
- [ ] 桌面端仍保持“正文 + 侧栏目录”阅读体验(目录可用)。

## 4. 降级与稳定性(通用)

- [ ] `prefers-reduced-motion` 下页面可用，且循环动效明显减少。
- [ ] 控制台无持续报错(进入/滚动/切换/hover 过程中无刷屏错误)。
