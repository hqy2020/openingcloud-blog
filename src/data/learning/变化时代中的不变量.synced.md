# 变化时代中的不变量

## 定义
> 在技术快速迭代、信息爆炸的时代，**识别和把握"不变量"是建立稳定认知框架的核心能力**。不变量是那些跨越时代、跨越工具、跨越方法论而始终保持有效的底层规律。

## 核心理念

在技术快速迭代、信息爆炸的时代，**识别和把握"不变量"是建立稳定认知框架的核心能力**。

不变量是那些**跨越时代、跨越工具、跨越方法论而始终保持有效的底层规律**。它们构成了我们思考和决策的坚实基座。

## 今日捕获的不变量

### 1. 输入质量决定论 ⭐⭐⭐⭐⭐
**表述**：你的输入质量决定了你的输出质量

**应用场景**：
- AI提示词编写
- 创意任务描述
- 问题咨询方式
- 学习资料选择

**实践原则**：
- 宁可多花时间思考如何问，也不急于要答案
- 用清晰、具体、上下文丰富的方式表达需求
- 将复杂问题分解为明确的小任务

### 2. 连接创造价值 ⭐⭐⭐⭐⭐
**表述**：知识是信息间的规律性连接，而非信息本身

**应用场景**：
- 笔记系统设计
- 知识管理体系
- 创意联想
- 问题解决

**实践原则**：
- 每个新信息都要建立至少一个连接
- 重视双向链接胜过完美分类
- 定期回顾和强化连接强度

## 不变量识别框架

### 识别标准
1. **跨时代性** - 在过去、现在、未来都成立
2. **跨工具性** - 独立于具体工具和平台
3. **跨领域性** - 适用于多个不同领域
4. **可验证性** - 能够通过实践验证
5. **指导性** - 能够指导具体行动

### 识别方法
1. **历史回溯** - 在历史中寻找重复模式
2. **跨域对比** - 比较不同领域的共同规律
3. **第一性原理** - 追问最底层的逻辑
4. **实践检验** - 在实际中验证有效性

## 实践建议

### 立即行动
- [ ] 建立"不变量"收集清单
- [ ] 定期回顾和验证已识别的不变量
- [ ] 在每个决策点问：这里的不变量是什么？

### 深化方向
- **领域不变量研究** - 在专业领域寻找不变规律
- **跨域不变量映射** - 探索不同领域间的共同规律
- **不变量工具化** - 将不变量转化为可执行的工具和方法
